# browser.cpp Privacy Policy

**browser.cpp** is a Chromium-family browser extension for Chrome, Edge, Brave,
and Chromium. It does not collect, store, or share user data.

The extension runs locally in the browser, stores workspace/session state in the
browser's extension storage area, and only reads or writes local files after the
user explicitly grants file or folder access through the browser's native file
picker prompts.
